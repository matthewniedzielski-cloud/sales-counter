
Sales Counter v2 — licznik sprzedaży (PWA) w stylu Google Flat Design

Funkcje:
- Szybkie +1 po tapnięciu, cofnięcie ostatniej akcji.
- Tryb odejmowania po przytrzymaniu + opcjonalny PIN.
- Produkty edytowalne (nazwa/kolor), zapisywane w localStorage.
- Historia dni (sumy + rozbicie).
- Eksport CSV (dzień / wszystkie dni), import CSV (date,product,count -> na dziś).
- Przełącznik dużych kafelków do pracy w rękawiczkach.
- Działa offline (PWA).

Instalacja (bez kodowania):
1) Wgraj folder na hosting statyczny (GitHub Pages / Netlify / Vercel).
2) Wejdź z telefonu, wybierz „Dodaj do ekranu głównego”.

Uwaga: Dane są lokalne w przeglądarce (localStorage). Aby przenieść na inne urządzenie, użyj eksportu/importu CSV.
